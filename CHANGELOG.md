## [1.0.0] - 2025-12-25

### Added
- Initial release of Step Box plugin
- `/step` command - Analyze project and generate workflow visualization
- `/step-update` command - Update existing documentation
- ASCII box diagram generation with Unicode characters
- Support for multiple project types (Node.js, Python, Go, etc.)
- Automatic timestamp tracking
- Custom section preservation with `<!-- CUSTOM -->` markers
- Detailed step documentation with file locations and dependencies

### Features
- Beautiful ASCII box visualization
- Automatic project structure analysis
- Entry point detection
- Dependency tracking
- File structure tree generation
